## 9. What Is Mock or Limited

These aspects are intentionally limited; the project is honest about them:

| Limitation | Detail |
|------------|--------|
| OCR | Mock only — no real image recognition |
| AI analysis | Mock only — no real AI/LLM processing |
| Real providers | None enabled — Qwen OCR/Analysis both gated behind env vars |
| Production readiness | `productionReady: false` |
| Allergy safety | No guarantee — informational only, not a diagnostic tool |
| Medical advice | None — not for health decision-making |
| TestFlight / App Store | Not released — requires macOS + Apple Developer membership |
| Backend | Render free tier, mock-only, no cold-start mitigation |
| API keys | None stored in Flutter or committed to repo |
