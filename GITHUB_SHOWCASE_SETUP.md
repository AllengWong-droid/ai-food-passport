### Pinned Caption Limitation

GitHub pinned repository cards **do not support** a separate editable custom caption in the GitHub profile UI. The card displays:

- Repository name (from the repo URL)
- Repository About description (from the About section configured above)
- Primary language color dot
- Star count
- Fork count

This is an **accepted GitHub UI limitation** — not a configuration gap or missing work. The project's public-facing description is controlled through:

1. **Repository About description** — visible on the pinned card
2. **README.md** — expanded view when the card is clicked
3. **Repository Topics** — aid GitHub's discovery algorithm

All three of these are properly configured and accurate.
