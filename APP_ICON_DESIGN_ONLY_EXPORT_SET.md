## 1. Purpose

This export set is **design-only** and serves the following purposes:

- **Preview future app icon sizes** — visualize how the clean square master will appear at all standard iOS icon dimensions
- **Design review** — enable stakeholder review of the icon at actual sizes before any technical implementation
- **NOT the final iOS `AppIcon.appiconset`** — these exports are not configured for Xcode asset catalog use
- **NOT applied to the app** — no Flutter code, iOS configuration, or build system references these files
- **Portfolio/demo reference** — provide sized exports for documentation and showcase purposes

The exports are generated from the **clean square master source** (Phase 22E) which resolves the Phase 22C baked-in rounded corner issue.

## Appendix A: Generation Command

The export set was generated using Python Pillow:

```python
from PIL import Image
import os

src = 'design/app-icon/source/ai-food-passport-clean-square-master.png'
out_dir = 'design/app-icon/export/design-only'
os.makedirs(out_dir, exist_ok=True)

img = Image.open(src)
sizes = [1024, 512, 180, 167, 152, 120, 87, 80, 76, 60, 58, 40, 29, 20]

for s in sizes:
    out_path = f'{out_dir}/icon-{s}.png'
    resized = img.resize((s, s), Image.LANCZOS)
    resized.save(out_path, 'PNG')
```

**Resampling method**: Lanczos (highest quality downsampling)  
**Pillow version**: 12.2.0  
**Python version**: 3.13.12 (managed)
